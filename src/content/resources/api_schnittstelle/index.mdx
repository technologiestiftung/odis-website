## Mit APIs Daten automatisiert bereitstellen und die Datennutzung erleichtern

Es gibt eine Vielzahl an Optionen, Daten als Open Data im Berliner Datenportal bereitzustellen. Eine Variante ist die auotmatisierte Bereitstellung über eine Programmierschnittstelle (API). Sie bietet sich insbesondere bei Daten an, die häufig aktualisiert werden – etwa wöchentlich, täglich oder noch öfter. 

Die Vorteile liegen auf der Hand: Durch eine Schnittstelle, beispielsweise zu einem Fachverfahren, müssen die Daten nicht manuell im Datenportal aktualisiert werden. APIs sind zudem bei großen Datenmengen oder einem gesamten Datenkatalog empfehlenswert, da die Daten trotzdem für den oder die Nutzer:in verarbeitbar bleiben.

Gerade im Hinblick auf das Thema Datensouveränität fördern APIs die Kontrolle über Daten und fördern den Datenaustausch innerhalb der Verwaltung.
